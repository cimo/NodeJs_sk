NodeJs sk

Starter kit code for web app. Create in a few steps the complete NodeJs project.

|:---|

| NodeJs version: 14.16.1 |

## Instructions:
1) Write on terminal:

         cd /home/cimo/nodejs_sk/root
         sudo cp src/Config.dist.ts src/Config.ts
         npm install
         npm run execute
